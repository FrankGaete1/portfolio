Financial Applications

Role

Lead UI/UX Architect @ Broadridge Financial Solutions, 2011-2016

Responsibilities

  • Analyze business requirements and UX research to develop design concepts via sketches, wireframes, high fidelity mockups and HTML-CSS-jQuery prototypes
  • Create, maintain and communicate UI/UX policies regarding design; coding; design/development approaches; supported browsers; cross browser testing; responsive web design; HTML email and accessibility standards
  • Provide creative and technical direction for the UI/UX team and all internal and external application projects both legacy and new; desktop and mobile, single applications and application platforms
  • Work with various financial application teams including tax, transfer agency, shareholder, banks, brokers, shared data services, corporate governance, proxy voting, mutual funds, universal e-delivery and others
  • Work with executives; business managers; technical managers; programmers; database architects, business analysts and other stakeholders to successfully meet UI/UX requirements
  • Collaborate directly with software engineers to successfully integrate UI/UX design and coding elements into Java and .Net environments using WebSphere and Visual Studio
  • Promote proficiency with current tools/languages via research and adoption of new tools/languages/approaches such as Adobe XD

Technology

  • .Net and Java environments
  • Visual Studio and Websphere
  • Liferay portal framework
  • JavaSript, jQuery, AngularJS 1.x and 2.0
  • HTML, CSS
  • Perforce
  • Adobe Photoshop, XD, Acrobat, InDesign, Illustrator
  • Waterfall and Agile Scrum
